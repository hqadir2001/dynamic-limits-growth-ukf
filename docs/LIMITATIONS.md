# Limitations

- The model is semi-structural and model-conditional.
- Parameters are calibrated defaults, not policy-invariant structural estimates.
- UKF filtering is not causal identification.
- Counterfactuals are fixed-shock within-model comparisons, not causal policy effects.
- Historical decomposition is accounting/model attribution, not external causal attribution.
- Results depend on data construction, observables, model specification, and noise assumptions.
